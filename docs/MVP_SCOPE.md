# Damso MVP Scope

## 서비스 목적

Damso(담소)는 자녀가 부모님에게 AI 인터뷰 질문을 보내고, 부모님이 스마트폰으로 답변하면 영상과 텍스트가 가족 다이어리와 회고록으로 기록되는 웹앱이다. 이 레포는 Damso 백엔드 전용 레포이며, 프론트엔드는 별도 레포에서 관리한다.

## 사용자

- 자녀: 부모님에게 질문을 보내고 답변, 다이어리, 회고록을 확인한다.
- 부모님: 스마트폰에서 초대 코드 또는 공유 링크로 접근해 질문에 답변한다.
- 가족 구성원: 허용된 범위 안에서 가족 다이어리와 회고록을 열람한다.

## 포함 기능

- 온보딩
- 카카오 로그인 진입
- 역할 선택
- 가족 초대 코드
- 질문 목록
- 질문 보내기
- 답변 기록
- AI 분석 상태
- 다이어리 목록 및 상세
- 회고록 결과
- 공유 링크

## 제외 기능

- 공유 링크
- 결제
- 관리자 대시보드
- 공개 커뮤니티
- 댓글
- 좋아요
- 팔로우
- 복잡한 영상 편집
- PDF 내보내기

## 구현 순서

1. FastAPI 앱, 설정, 헬스 체크, 테스트 기반을 만든다.
2. 인증 진입과 사용자 역할 선택 흐름을 정의한다.
3. 가족 생성, 초대 코드, 구성원 연결 흐름을 구현한다.
4. 질문 목록과 질문 보내기 API를 구현한다.
5. 답변 기록 저장과 업로드 메타데이터 흐름을 구현한다.
6. AI 질문 생성, 답변 요약, 분석 상태 처리를 붙인다.
7. 다이어리 목록/상세와 회고록 결과 API를 구현한다.

DB 테이블, SQLAlchemy 모델, Alembic migration은 ERD 확정 후 만든다.
