# Damso MVP Scope

## 서비스 목적

Damso(담소)는 자녀가 부모님에게 AI 인터뷰 질문을 보내고, 부모님이 스마트폰으로 영상 답변을 하면 AI가 가공한 영상 클립(네컷 그리드)으로 기록되는 웹앱이다. 이 레포는 Damso 백엔드 전용 레포이며, 프론트엔드는 별도 레포에서 관리한다.

## 사용자

- 자녀: 부모님에게 질문을 보내고 영상 클립을 확인한다.
- 부모님: 스마트폰에서 초대 코드로 접근해 질문에 영상으로 답변한다.
- 가족 구성원: 허용된 범위 안에서 가족의 영상 클립을 열람한다.

## 포함 기능

- 온보딩
- 카카오 로그인 진입
- 역할 선택
- 가족 초대 코드
- 질문 목록
- 질문 보내기
- 답변 기록
- 영상 클립 (네컷 그리드, 상세)

## 제외 기능

- 공유 링크(`/s/:shareSlug`)
- 결제
- 관리자 대시보드
- 공개 커뮤니티
- 댓글
- 좋아요
- 팔로우
- 복잡한 영상 편집
- PDF 내보내기

## 구현 순서

1. FastAPI 앱, 설정, 헬스 체크, 테스트 기반을 만든다.
2. 인증 진입과 사용자 역할 선택 흐름을 정의한다.
3. 가족 생성, 초대 코드, 구성원 연결 흐름을 구현한다.
4. 질문 목록과 질문 보내기 API를 구현한다.
5. 답변 기록 저장과 업로드 메타데이터 흐름을 구현한다.
6. AI 질문 생성을 붙인다.
7. 답변 영상 AI 가공(영상 클립 생성)과 네컷 그리드, 클립 상세 API를 구현한다.

DB 테이블, SQLAlchemy 모델, Alembic migration은 ERD 확정 후 만든다.
