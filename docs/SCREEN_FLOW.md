# Damso Screen Flow

## 온보딩

- 사용자 행동: 서비스 소개 화면을 보고 카카오 로그인으로 진입한다.
- API 후보: `GET /health`, `GET /api/v1/auth/kakao/login-url`
- 저장 데이터: 없음. 로그인 완료 후 사용자 식별 정보 저장 예정.

## 카카오 로그인 진입

- 사용자 행동: 카카오 계정으로 인증하고 서비스로 돌아온다.
- API 후보: `POST /api/v1/auth/kakao/callback`, `GET /api/v1/users/me`
- 저장 데이터: OAuth provider, provider user id, 사용자 프로필, 세션 또는 토큰 정보.

## 역할 선택

- 사용자 행동: 자식, 엄마, 아빠 중 하나의 역할을 선택한다.
- API 후보: `PATCH /api/v1/users/me/role`
- 저장 데이터: user id, role, role selected at.

## 가족 초대 코드

- 사용자 행동: 자녀가 가족을 만들고 초대 코드를 공유한다. 부모님은 초대 코드로 가족에 합류한다.
- API 후보: `POST /api/v1/families`, `POST /api/v1/families/invite-code`, `POST /api/v1/families/join`
- 저장 데이터: family id, invite code hash, code expiry, member role, membership status.

## 질문 목록

- 사용자 행동: 보낼 수 있는 질문 목록과 이전 질문 상태를 확인한다.
- API 후보: `GET /api/v1/questions`, `GET /api/v1/questions/{question_id}`
- 저장 데이터: question id, category, question text, generated by, status.

## 질문 보내기

- 사용자 행동: 자녀가 부모님에게 질문을 선택하거나 AI 추천 질문을 보낸다.
- API 후보: `POST /api/v1/questions/send`
- 저장 데이터: recipient id, sender id, question id, sent at, answer due status.

## 답변 기록

- 사용자 행동: 부모님이 스마트폰에서 영상으로 답변한다.
- API 후보: `POST /api/v1/answers`
- 저장 데이터: answer id, question send id, user id, family id, video origin url, video asset metadata, submitted at.

## 네컷 그리드

- 사용자 행동: AI 처리가 끝난 답변 영상을 날짜 + 가족 단위로 묶은 네컷 그리드를 확인한다.
- API 후보: `GET /api/v1/clips`
- 저장 데이터: 없음. `answers`를 `family_id`, `DATE(created_at)` 기준으로 조회한 결과.

## 컷 상세

- 사용자 행동: 그리드에서 컷을 탭해 바텀시트 또는 상세에서 영상 재생, 명대사, 요약을 확인한다.
- API 후보: `GET /api/v1/answers/{answer_id}/clip`
- 저장 데이터: video_clip id, answer id, thumbnail url, hls url, transcript, title, quote, summary, emotion tags.
