# Damso Screen Flow

## 온보딩

- 사용자 행동: 서비스 소개 화면을 보고 카카오 로그인으로 진입한다.
- API 후보: `GET /health`, `GET /api/v1/auth/kakao/login-url`
- 저장 데이터: 없음. 로그인 완료 후 사용자 식별 정보 저장 예정.

## 카카오 로그인 진입

- 사용자 행동: 카카오 계정으로 인증하고 서비스로 돌아온다.
- API 후보: `POST /api/v1/auth/kakao/callback`, `GET /api/v1/users/me`
- 저장 데이터: OAuth provider, provider user id, 사용자 프로필, 세션 또는 토큰 정보.

## 역할 선택

- 사용자 행동: 자녀 또는 부모 역할을 선택한다.
- API 후보: `PATCH /api/v1/users/me/role`
- 저장 데이터: user id, role, role selected at.

## 가족 초대 코드

- 사용자 행동: 자녀가 가족을 만들고 초대 코드를 공유한다. 부모님은 초대 코드로 가족에 합류한다.
- API 후보: `POST /api/v1/families`, `POST /api/v1/families/invite-code`, `POST /api/v1/families/join`
- 저장 데이터: family id, invite code hash, code expiry, member role, membership status.

## 질문 목록

- 사용자 행동: 보낼 수 있는 질문 목록과 이전 질문 상태를 확인한다.
- API 후보: `GET /api/v1/questions`, `GET /api/v1/questions/{question_id}`
- 저장 데이터: question id, category, question text, generated by, status.

## 질문 보내기

- 사용자 행동: 자녀가 부모님에게 질문을 선택하거나 AI 추천 질문을 보낸다.
- API 후보: `POST /api/v1/questions/send`
- 저장 데이터: recipient id, sender id, question id, sent at, answer due status.

## 답변 기록

- 사용자 행동: 부모님이 스마트폰에서 영상 또는 텍스트로 답변한다.
- API 후보: `POST /api/v1/answers`, `GET /api/v1/answers/{answer_id}`
- 저장 데이터: answer id, question id, respondent id, text answer, video asset metadata, submitted at.

## AI 분석 상태

- 사용자 행동: 답변 제출 후 요약과 분석 진행 상태를 확인한다.
- API 후보: `GET /api/v1/answers/{answer_id}/analysis`, `POST /api/v1/answers/{answer_id}/analysis`
- 저장 데이터: analysis status, summary, keywords, model metadata, error reason.

## 다이어리 목록/상세

- 사용자 행동: 가족 다이어리 목록과 개별 기록을 확인한다.
- API 후보: `GET /api/v1/diaries`, `GET /api/v1/diaries/{diary_id}`
- 저장 데이터: diary id, family id, linked answer ids, title, preview, created at.

## 회고록 결과

- 사용자 행동: 누적된 답변으로 생성된 회고록 결과를 확인한다.
- API 후보: `POST /api/v1/memoirs`, `GET /api/v1/memoirs`, `GET /api/v1/memoirs/{memoir_id}`
- 저장 데이터: memoir id, family id, source diary ids, generated text, generation status.

## 공유 링크

- 사용자 행동: 가족 다이어리 또는 회고록을 외부 가족에게 공유한다.
- API 후보: `POST /api/v1/share-links`, `GET /api/v1/share-links/{token}`
- 저장 데이터: token hash, target type, target id, expiry, permission scope.
